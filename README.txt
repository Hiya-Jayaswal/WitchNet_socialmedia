# WitchNet - Mystical Social Media Platform

Welcome to WitchNet, a revolutionary social media platform designed for modern-day enthusiasts of the arcane arts. WitchNet bridges the gap between technology and mysticism, empowering users to connect, share, and explore the mystical world.

## Features
- Content Management System (CMS) for publishing blogs, articles, and multimedia.
- Social Media Platform for wizards to connect, share knowledge, and engage with the community.
- Real-time post feed, notifications, and messaging capabilities.
- Smooth user experience with intuitive interface design.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Templating Engine:** Handlebars

## Getting Started
1. Clone this repository.
2. Install dependencies using `npm install`.
3. Start the server using `node index.js`.in src directory
4. Visit `http://localhost:3000` in your browser to access WitchNet.

## Folder Structure
- **public:** Contains static assets like CSS and client-side JavaScript files.
- **views:** Contains Handlebars templates for server-side rendering.
- **routes:** Contains Express.js route handlers.
- **models:** Contains Mongoose models for interacting with the MongoDB database.
- **config:** Contains configuration files for Express.js and other modules.
